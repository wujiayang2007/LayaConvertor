# LayaConvertor

本项目将帮助您将 Laya 工程中的资源转换为 Unity 工程的资源

### 如何使用
克隆本存储库或者下载发布的UnityPackage导入到您的Unity项目中

### 功能
1.将未压缩的Laya模型转换为Unity模型(支持LayaModel0301~LayaModel0502)
